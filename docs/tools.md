# ツール

AWS EC2構築に役立つツールとコマンドリファレンスを提供します。

---

## 📚 コンテンツ

### コマンドリファレンス

よく使うAWS CLIコマンドを簡単に生成できるツールです。

* [コマンドリファレンス](command-reference) - CloudFormation、EC2などのコマンド生成ツール

---

## 使い方

1. 各ツールページにアクセス
2. 必要な情報を入力
3. 自動生成されたコマンドをコピーして使用

---

## 特徴

* ✅ パラメータを入力するだけでコマンド生成
* ✅ タイプミスを防止
* ✅ よく使うコマンドを一覧で管理
